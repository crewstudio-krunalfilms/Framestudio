Integration guide (face-api.js — free & open source)
Add to your real app
1
Include library:
<script src="https://cdn.jsdelivr.net/npm/face-api.js"></script>
2
Load models once:
await faceapi.nets.ssdMobilenetv1.loadFromUri('/models')\nawait faceapi.nets.faceRecognitionNet.loadFromUri('/models')
3
Compare selfie to each photo:
const dist = faceapi.euclideanDistance(selfieDesc, photoDesc)\nif(dist < 0.5) showPhoto() // match!
4
All runs in the client's browser. Zero server cost. Zero privacy concern.
